# Smartstream

Smartstream ist eine modulare Windows-Software zur Konfiguration und Steuerung eines anpassbaren Eingabesystems.

Tasten, Drehgeber, Schieberegler und weitere Bedienelemente können mit Aktionen belegt werden. Smartstream ist als flexible Alternative zu fest vorgegebenen Bedienkonsolen gedacht.

> **Hinweis:** Smartstream befindet sich derzeit in der Testphase. Fehler, unvollständige Funktionen und Änderungen am Verhalten sind zu erwarten. Die Menschheit nennt so etwas „Vorabversion“, damit Abstürze wenigstens organisatorisch korrekt eingeordnet sind.

## Installation

1. Lade `SmartstreamSetup.exe` über den Download-Knopf oben herunter.
2. Öffne die heruntergeladene Datei.
3. Folge den Schritten des Installationsprogramms.
4. Starte Smartstream über die erstellte Verknüpfung.

Windows oder ein Browser kann bei neuen und noch wenig verbreiteten Programmdateien eine Sicherheitswarnung anzeigen. Prüfe vor dem Start, dass die Datei aus dem offiziellen Smartstream-Repository stammt.

## Aktueller Status

Smartstream wird aktiv entwickelt und ist noch nicht als stabile Endversion eingestuft.

Derzeit liegt der Schwerpunkt auf:

- Erkennung angeschlossener Module;
- Verarbeitung von Tasten-, Drehgeber- und Schiebereglerereignissen;
- lokalen Profilen und Aktionszuweisungen;
- zuverlässiger Kommunikation zwischen Hardware und Windows-Anwendung;
- automatischer Aktualisierung der Anwendung.

## Systemvoraussetzungen

- Windows 10 oder Windows 11;
- 64-Bit-System;
- Internetverbindung für Download und automatische Aktualisierungen;
- unterstützte Smartstream-Hardware für die vollständige Nutzung.

Die Anwendung wird als eigenständige Windows-Programmdatei veröffentlicht. Eine separate .NET-Installation sollte für die veröffentlichte Version grundsätzlich nicht erforderlich sein.

## Automatische Aktualisierungen

Smartstream prüft beim Start automatisch über GitHub, ob eine neuere Version verfügbar ist.

Neue Versionen werden automatisch heruntergeladen und installiert. Eine Abschaltung dieser Funktion ist nicht vorgesehen.

Die automatische Aktualisierung ist Bestandteil des vorgesehenen Betriebs. Das Umgehen, Manipulieren oder dauerhafte Blockieren der Aktualisierung verstösst gegen die [Nutzungsbedingungen](NUTZUNGSBEDINGUNGEN.md).

## Datenschutz

Smartstream sammelt derzeit keine Telemetrie, Nutzungsstatistiken oder automatischen Fehlerberichte.

Konfigurationen und Aktionszuweisungen werden grundsätzlich lokal auf dem Computer gespeichert.

Für die automatische Aktualisierung verbindet sich der Launcher mit GitHub. Dabei verarbeitet GitHub technisch notwendige Verbindungsdaten.

Weitere Informationen stehen in der [Datenschutzerklärung](DATENSCHUTZERKLAERUNG.md).

## Nutzung

Smartstream kann unter anderem folgende Aktionen unterstützen:

- Tastenkombinationen ausführen;
- Medien steuern;
- Lautstärke verändern;
- Programme oder Dateien öffnen;
- Texte eingeben;
- Profile und Seiten wechseln;
- unterstützte Drittanbieterprogramme steuern.

Der genaue Funktionsumfang hängt von der installierten Version ab.
